# docker compose 

다중 컨테이너 애플리케이션을 정의하고 실행하기 위한 도구이다.

모든 환경에서 동작한다. 

# docker compose 의 이점

1. 단순화된 제어

yaml 파일을 통해서 다중 컨테이너 애플리케이션을 정의하고 관리할 수 있다.

2. 효율적인 협업

config file은 쉽게 공유할 수 있어서 협업이 쉽다.

3. 개발속도 향상

사용된 구성을 캐싱하기 때문에 빠르게 환경 변경 가능

4. 환경간 이식성

변수를 지원하는 것을 통해 상황에 맞게 정의가능

5. 커뮤니티 지원

커뮤니티 쩔음
